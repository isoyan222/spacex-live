SPACEX LIVE とは？

SpaceX が公開しているリアルタイムデータを1つの画面で完全網羅。Starlink 衛星の軌道追跡から打ち上げカウントダウン、株価まで。宇宙開発の今を、誰でも無料で楽しめるダッシュボードです。

🚀 ダッシュボードを使う
できること
🌍
3D 地球儀 + Starlink 追跡
リアルタイムで地球を周回する Starlink 衛星 6,800 機以上を3D地球儀上で可視化。ドラッグで地球を自由に回転できます。
🚀
打ち上げカウントダウン
次の SpaceX ロケット打ち上げまでの残り時間をリアルタイムで表示。打ち上げ60分前には生放送バナーが自動表示されます。
🛸
ISS リアルタイム追跡
国際宇宙ステーション（ISS）の現在位置・高度・速度をリアルタイム更新。「地球儀で追う」ボタンで ISS の上空を確認できます。
📈
SPCX 株価チャート
2026年6月に上場した SpaceX（SPCX）の株価を IPO からの推移で表示。TSLA・RKLB・ASTS・LUNR などの関連株も確認できます。
🌙
月・火星ビュー
NASA Artemis の月面着陸候補地や、Starship の火星ミッション目標地点を3Dビューで表示。宇宙探査の目標を視覚化します。
🚗
Tesla Roadster 追跡
2018年に宇宙へ打ち上げられたイーロン・マスクの Tesla Roadster の現在位置・地球からの距離・速度を表示します。
🎬
名場面アーカイブ
Falcon 9 史上初着陸・Falcon Heavy 双子ブースター着陸・Starship Super Heavy キャッチなど、SpaceX の歴史的瞬間を動画で振り返れます。
🔔
打ち上げ通知
次の打ち上げの30分前・10分前にブラウザ通知を受け取れます。見逃したくない打ち上げも安心。スマホにアプリとして追加も可能（PWA 対応）。
📡 使用データソース
SPACEX API
SpaceX Public API（r-spacex/SpaceX-API）
打ち上げ情報・ロケットスペック・Starlink 衛星位置・乗組員データ・Roadster 軌道データ
NASA
NASA Open Notify API
ISS リアルタイム位置・速度・高度データ、現在の ISS 乗組員情報
FINANCE
Yahoo Finance（非公式）
SPCX・TSLA・RKLB・ASTS・LUNR の株価データ
MAP
Globe.gl / Three.js
3D 地球儀のレンダリング（WebGL）
YOUTUBE
YouTube（Google LLC）
名場面アーカイブの動画埋め込み
